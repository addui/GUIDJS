# GUID.JS

A simple script for generating a Globally Unique Identifiers (GUID) in JavaScript.

## Create a new GUID

```JavaScript
var myGuid = GUID();
```

## Get a list of all GUIDs

```JavaScript
GUID.list
```

## Check if a GUID exists

```JavaScript
GUID.exists("885b8819-5bb3-467b-9e6b-62ca997b55a5");
```
